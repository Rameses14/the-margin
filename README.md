# The Margin

A fortnightly magazine of opinion, reporting, and verse — written entirely by students.

This is the website: a set of plain HTML/CSS pages (no build step, no server needed).
Open `index.html` in a browser to view it locally, or see **DEPLOY.md** to put it online
for free with a public link.

## What's here
- **Reader pages** — homepage (`index.html`), article pages, verse, the Arts section,
  Issues, Topics, Writers, Saved, and About. Readers need no login.
- **Editor's suite** — `login.html` → dashboard, planning, drafts, issues, topics,
  writers/access, settings, and the writing view. *Demo login:* `editor@themargin.test`
  / `demo1234`. (Front-end demo — nothing is saved yet.)
- `styles.css` — styling for every page.

## Status
A demo mockup. The masthead name "The Margin" is a placeholder and there's no school
branding yet. Sample articles are placeholders for real student writing.
